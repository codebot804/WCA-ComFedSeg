# Datasets

Dataset loaders and split builders for binary segmentation experiments.

Implemented loaders:

- `busi.py`: BUSI breast ultrasound image-mask pairs.
- `bus_uclm.py`: BUS-UCLM breast ultrasound lesion segmentation.
- `kvasir_seg.py`: Kvasir-SEG polyp segmentation.
- `synthetic_segmentation.py`: lightweight synthetic segmentation data for pipeline checks.

Raw datasets are not included in this repository. See `data/README.md` for the expected local layout.
