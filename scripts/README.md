# Scripts

Utility scripts retained for the public reproducibility package.

- `validate_real_datasets.py`: check dataset discovery and image-mask pairing.
- `visualize_real_dataset_samples.py`: generate qualitative sample visualizations.
- `validate_phase5e_hard_noniid_splits.py`: inspect hard non-IID client distributions.
- `validate_phase5o_bus_uclm_loader.py`: validate BUS-UCLM loader behavior.
- `validate_phase5p_ultrasound_splits.py`: summarize ultrasound split statistics.
- `compare_phase5m_moderate_multiseed.py`: aggregate BUSI/Kvasir moderate multi-seed summaries.
- `compare_phase5k_core_hard_multiseed.py`: aggregate hard non-IID boundary summaries.
- `compare_phase5q_bus_uclm_moderate_pilot.py`: summarize BUS-UCLM moderate pilot results.
- `compare_phase5s_bus_uclm_wca_bn_scoped_pilot.py`: summarize WCA+BN diagnostic results.
- `diagnose_phase5l_busi_wca_prox_instability.py`: analyze BUSI hard non-IID WCA+FedProx instability.
- `diagnose_phase5r_bus_uclm_failure.py`: analyze BUS-UCLM WCA-Comm failure behavior.
